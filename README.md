# Python_lessons

## Как запускать тесты и просматривать отчеты

### Установка необходимых зависимостей

Перед началом убедитесь, что установлены все необходимые библиотеки:
pip install -r requirements.txt

Или, если файла requirements.txt нет, выполните:
pip install selenium pytest allure-pytest

Также убедитесь, что у вас установлен драйвер для браузера (например, ChromeDriver или GeckoDriver для Firefox).

Запуск тестов и генерация отчета:

Запустите тесты командой pytest, указав папку для результатов Allure:
pytest --alluredir=allure-results

Это выполнит все тесты и сохранит результаты в папке allure-results.

После успешного завершения тестов откройте отчет с помощью сервера:
allure serve allure-results

Эта команда запустит локальный сервер и автоматически откроет окно браузера с интерактивным отчетом. После закрытия окна сервер остановится.

Просмотр сформированного отчета:

Отчет позволяет просматривать все прогоны, шаги, скриншоты и логи.
В нем удобно анализировать причины ошибок и проверять выполнение каждого теста.

Для статического сохранения отчета выполните:
allure generate allure-results -o allure-report --clean

Чтобы открыть сохраненный отчет в браузере, используйте:
allure open allure-report

Теперь вы готовы запускать автоматические тесты, смотреть красивые отчеты и анализировать результаты!)
